## For Ubuntu 26.04 Tablet Mode

Rotation mode will not work without this workaround applied at this time.

### Workaround for a 26.04 tablet mode bug

You must [update your installation as described here](https://github.com/FrameworkComputer/linux-docs/blob/main/framework12/Ubuntu-26.04.md#this-is-for-ubuntu-2604-on-framework-laptop-12-only), otherwise tablet rotatioon mode will not work.

Once updated, simply fold back your screen so it lays flat fully foldeed, then rotate your laptop.
